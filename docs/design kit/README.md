# Work Hours Follow — UI Guidelines

This file provides the concise rules an AI agent should keep in context while editing UI code.

- Match the supplied design references.
- Use deep navy as the main background.
- Use warm cream cards for primary records and totals.
- Use red only for primary action, selected tab, and destructive emphasis.
- Use muted gold for earnings, supporting icons, and refined emphasis.
- Use serif typography for titles and major totals.
- Use SF Pro for controls, body content, captions, and navigation.
- Use 24-point horizontal screen padding.
- Use 16-point card padding.
- Use 12-point spacing between cards.
- Use 16-point standard card radius.
- Prefer borders and subtle separation over heavy shadows.
- Use native SF Symbols.
- Keep all essential actions at least 44×44 points.
- Do not use gradients, glassmorphism, charts, or decorative animations.
- Do not create a start/end shift interface.
- Duration input must be hours and minutes.
- The product remains dark-mode-first.
- Accessibility and native iOS behavior take priority over literal pixel copying.
